# 🐕 乐乐的识字乐园

专为3-5岁儿童设计的汉字学习应用，汪汪队主题风格。

## 功能

- 🔤 **学汉字** - 卡片学习，点击朗读
- 🖼️ **看图选字** - 看图片选汉字
- ❓ **汉字测验** - 选择题测验
- 🎯 **汉字配对** - 汉字和拼音配对
- 🔨 **打地鼠** - 快速找汉字
- ⏱️ **限时挑战** - 60秒计时答题

## 下载APK

点击 [Actions](../../actions) → 选择最新的构建 → 下载 `lele-chinese-apk`

## 本地运行

```bash
pip install kivy
python chinese_app_pydroid.py
```

## 技术栈

- Python 3.11
- Kivy 2.x
- Android TTS 语音
