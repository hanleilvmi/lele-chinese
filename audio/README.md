# 🐕 汪汪队语音包使用说明

## 文件夹结构

```
audio/
├── praise/          # 表扬语（答对时播放）
├── encourage/       # 鼓励语（答错时播放）
├── effects/         # 音效
│   ├── click.mp3    # 按钮点击音效
│   ├── correct.mp3  # 答对音效
│   ├── wrong.mp3    # 答错音效
│   ├── levelup.mp3  # 升级音效
│   └── star.mp3     # 获得星星音效
└── background.mp3   # 背景音乐（可选）
```

## 需要准备的音频

### praise/ 表扬语（建议8-10个）
录制或截取以下内容的音频：
1. "汪汪队，出动！答对啦！"
2. "没有困难的工作，只有勇敢的狗狗！"
3. "太棒了，你是最勇敢的狗狗！"
4. "耶！任务完成！"
5. "狗狗们，做得好！"
6. "给你一个大大的赞！"
7. "汪汪汪，你真棒！"
8. "莱德队长为你骄傲！"

### encourage/ 鼓励语（建议5个）
1. "没关系，汪汪队永不放弃！"
2. "加油，勇敢的狗狗不怕困难！"
3. "再试一次，你一定行！"
4. "别担心，汪汪队来帮你！"
5. "狗狗们，我们再来一次！"

### effects/ 音效（可选）
- click.mp3 - 按钮点击音效（建议0.1-0.3秒）
- correct.mp3 - 答对音效（建议0.5-1秒）
- wrong.mp3 - 答错音效（建议0.5-1秒）
- levelup.mp3 - 升级音效（建议1-2秒）
- star.mp3 - 获得星星音效（建议0.3-0.5秒）

### background.mp3 背景音乐（可选）
- 轻快的儿童音乐
- 建议音量较低，不影响语音播放
- 程序会循环播放

## 获取音频的方法

### 方法1：自己录制
用手机或电脑录音，模仿汪汪队的语气录制上述内容

### 方法2：从动画片截取
1. 在B站/优酷找汪汪队动画片
2. 用录屏软件录制需要的片段
3. 用 Audacity（免费软件）剪辑成单独的音频文件
4. 导出为 mp3 格式

### 方法3：AI声音克隆（高级）
使用 GPT-SoVITS 等工具克隆莱德的声音

### 方法4：免费音效网站
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- 搜索 "game sound effects" 下载免费音效

## 文件格式要求
- 格式：mp3 或 wav
- 文件名：随意（praise/encourage），固定名称（effects）
- 时长：建议2-5秒（语音），0.1-2秒（音效）

## 使用方式
1. 将音频文件放入对应文件夹
2. 程序会自动检测并使用自定义音频
3. 在主菜单可以通过 🔊/🎵 按钮控制音效和背景音乐
4. 如果文件夹为空，程序会使用默认的 TTS 语音
