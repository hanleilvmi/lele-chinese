# 打包成 EXE 可执行文件

## 方法一：使用 PyInstaller（推荐）

### 1. 安装 PyInstaller
```bash
pip install pyinstaller
```

### 2. 打包主程序（单个EXE文件）
```bash
pyinstaller --onefile --windowed --name "乐乐学习乐园" --icon=icon.ico kids_learning_main.py
```

参数说明：
- `--onefile`: 打包成单个EXE文件
- `--windowed`: 不显示命令行窗口
- `--name`: 指定输出文件名
- `--icon`: 指定图标（可选，需要准备.ico文件）

### 3. 打包后的文件位置
```
dist/乐乐学习乐园.exe
```

### 4. 需要一起分发的文件
```
乐乐学习乐园/
├── 乐乐学习乐园.exe      # 主程序
├── audio/                # 音频文件夹
│   ├── praise/
│   ├── encourage/
│   └── effects/
├── voice_config.json     # 语音配置（可选）
└── 字库模板.txt           # 字库模板（可选）
```

## 方法二：一键打包脚本

运行下面的命令进行打包：

```bash
python build_exe.py
```

---

## 注意事项

1. **首次运行较慢**：打包后的EXE首次启动需要解压，会比较慢，之后就正常了

2. **杀毒软件误报**：部分杀毒软件可能误报，需要添加信任

3. **依赖库**：确保安装了所有依赖：
   ```bash
   pip install edge-tts pygame pypinyin pyinstaller
   ```

4. **文件大小**：打包后约 50-100MB（包含Python运行时）

5. **音频文件**：audio文件夹需要和EXE放在同一目录
